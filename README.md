# τ³ Assessment Run Bundle

Install the dependency, create `.env` with `OPENAI_API_KEY`, and run the commands in `results/tau3/assessment/RUNBOOK.md`.

For implementation or troubleshooting assistance, provide [AI_CONTEXT.md](AI_CONTEXT.md) with the error log.

Included input files:

- `results/tau3/annotation_cases/part-*.jsonl`
- `results/tau3/policy_packet/annotation_packet_private_key.csv`
The bundle produces `main_text_summary.json` for manuscript tables and retains detailed assessment/provenance outputs for later analysis.
